## Overview

The Pattern Language consists of 253 patterns organized into **36 sequences**. Each sequence is a coherent flow of related patterns that work together to create **emergent phenomena**‚Äîproperties that arise from the synergy of patterns rather than from any single pattern.

This guide helps you navigate through sequences to understand how patterns compose into living wholes.

## Understanding Sequences

### What is a Sequence?

A sequence is a carefully ordered group of patterns that:
- Progress from large-scale to small-scale (usually)
- Build upon each other contextually
- Create synergies between patterns
- Generate emergent properties beyond individual patterns

### How to Use Sequences

1. **Follow in Order**: Patterns within a sequence build on previous ones
2. **Notice Emergence**: Watch for properties that arise from combinations
3. **Make Connections**: See how patterns reinforce and complete each other
4. **Apply Gradually**: Implement patterns in sequence order when possible
